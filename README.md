📌 Projeto API RESTful (Estudo)
🧠 Sobre o projeto

Este projeto foi desenvolvido como parte do meu processo de aprendizado em desenvolvimento de APIs RESTful. O objetivo principal foi entender, na prática, como estruturar, construir e organizar uma API seguindo os princípios REST.

Trata-se de um projeto de estudo, criado para experimentar conceitos, boas práticas e ferramentas utilizadas no desenvolvimento backend.

🚀 Objetivos
Aprender como funciona uma API REST
Entender conceitos como:
Rotas (endpoints)
Métodos HTTP (GET, POST, PUT, DELETE)
Estrutura de requisições e respostas
Praticar organização de código
Explorar boas práticas no desenvolvimento backend
🛠️ Tecnologias utilizadas
Java
Spring Boot
db.migration
📚 Aprendizados

Durante o desenvolvimento deste projeto, pude aprender:

Como criar e organizar endpoints
Como lidar com dados e requisições HTTP
Noções de arquitetura de APIs
Tratamento de erros
Estruturação de um projeto backend
⚠️ Observações

Este projeto não tem fins comerciais e não está pronto para produção. Ele foi desenvolvido exclusivamente para fins de estudo e aprendizado.
